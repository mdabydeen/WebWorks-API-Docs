addEventListener

static void addEventListener(eventName : String, yourCallbackFunction : function)


Supported Platform(s)

 - BlackBerry 10

Description

Allows you to attach a callback function to an event. You can attach multuple callback functions to a single event.



Parameter Type Description
eventName String The event you want to attach the callback function to.
yourCallbackFunction function() The callback function that will execute when the event occurs.

removeEventListener

static void removeEventListener(eventName : String, yourCallbackFunction : function)


Supported Platform(s)

 - BlackBerry 10

Description

Allows you to remove a callback function from an event.



Parameter Type Description
eventName String The event you want to remove the callback function from.
yourCallbackFunction function() The callback function that will be removed from the event.
