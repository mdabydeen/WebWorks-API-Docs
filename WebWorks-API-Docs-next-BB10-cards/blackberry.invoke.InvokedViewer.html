InvokedViewer

The InvokedViewer object provides access to the viewer object that is invoked by the host application. This object cannot be created by the new keyword. It is generated only by blackberry.invoke.invokeViewer


Supported Platform(s)

- BlackBerry 10
- Ripple Emulator

send

static void send(message : String, [callback: function])


Supported Platform(s)

 - BlackBerry 10
 - Ripple Emulator

Description

Send a message to the viewer and set the callback when viewer responds to the message



Parameter Type Description
message String Describe the message that will be sent to viewer
callback function(message : String)
Optional 		Callback function that will be triggered when viewer responds to the message

message: Describe the message that is sent from the viewer that responds to the host's message

Code Example(s)

<script type="text/javascript">

// Set the new size of viewer window to 400 by 600
function viewerRespondToSave(message) {
    if (message == "saved") {
        alert("The content of viewer is saved.");
    } else {
        alert("Cannot save the content of viewer.");
    }
}

function sendSaveMessageToViewer() {
    // viewer object is stored from onInvokeViewerSuccess callback
    viewer.send("save", viewerRespondToSave);
}

</script>

Code Example(s)

<script type="text/javascript">

function onViewerClosed() {
    alert("The Viewer window was closed.");
}

function onViewerCanceled() {
    alert("The Viewer was canceled.");
}

function onInvokeViewerSuccess(viewer) {
    viewer.setPosition(0, 0);
    viewer.setVisibility(true);
    viewer.setZOrder(10);
    viewer.onClosed = onViewerClosed;
    viewer.onCanceled = onViewerCanceled;
}

function onInvokeViewerError(error) {
    alert("Invocation failed, error: " + error);
}

function invokeViewer() {
    // To invoke a Viewer, you need provide width and height fields
    blackberry.invoke.invokeViewer({
        target: "a.viewer.app", // The target should point to the "Viewer" target entry point of an application
        width: 600,
        height: 600
    }, onInvokeViewerSuccess, onInvokeViewerError);
}

</script>
