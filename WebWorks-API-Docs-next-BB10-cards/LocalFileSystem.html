requestFileSystem

void requestFileSystem(String : string, callback : successCallback, callback : errorCallback)


Supported Platform(s)

 - BlackBerry PlayBook 1.0+
 - BlackBerry 10
 - Ripple Emulator

Description

Requests the filesystem.



Parameter Type Description
String string to indicate the LocalFileSystem.type (ie. LocalFileSystem.PERSISTENT or LocalFileSystem.TEMPORARY)
callback successCallback() function that is invoked on successful request of a file system. Argument passed in is the FileSystem object
callback errorCallback() function for handling errors or when the request to obtain the file system is denied. Argument passed in is the FileError object

Code Example(s)

Main script:

<script type="text/javascript">
function onInitFs(fs) {
		console.log('Opened file system: ' + fs.name);
}

window.requestFileSystem(window.TEMPORARY, 5*1024*1024  onInitFs, errorHandler);    // 5*1024*1024 = 5MB


</script>

resolveLocalFileSystemURI

void resolveLocalFileSystemURI(Full : String, size : integer, callback : successCallback, Optional : errorCallback)


Supported Platform(s)

 - BlackBerry PlayBook 1.0+
 - BlackBerry 10
 - Ripple Emulator

Description

Requests the DirectoryEntry or FileEntry Object using local URI.



Parameter Type Description
Full String path and name of file
size integer (in bytes) the app will require for storage
callback successCallback() function that is invoked on successful request of a file system. Argument passed in is the FileSystem object
Optional errorCallback() callback for handling errors or when the request to obtain the file system is denied. Argument passed in is the FileError object

Code Example(s)

Main script:

<script type="text/javascript">

function onSuccess(fileEntry) {
		console.log(fileEntry.name);
}

window.resolveLocalFileSystemURI("file:///example.txt", onSuccess, onError);    * </script>
