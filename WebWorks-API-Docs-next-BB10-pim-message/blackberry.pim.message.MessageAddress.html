Supported Platforms

Parameters
type The type of the email MessageAddress; it is one of MessageAddress.TO, MessageAddress.CC and MessageAddress.BCC.
displayName The display name of the object.
email The email address of the object.

Code Example: 
    var MessageAddress = new MessageAddress(MessageAddress.TO, "Eric", "eric@somedomain.com");
