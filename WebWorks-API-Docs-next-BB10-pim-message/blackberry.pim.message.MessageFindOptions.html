Supported Platforms

Parameters
account Optional. This specifies the message account under which the search will be conducted. The default value is null, which means the search will be conducted under all the existing message accounts.
folder Optional. This specifies the message folder under which the search will be conducted. The default value is null, which means the search will be conducted under all the message folders.
filter Optional. A MessageFindFilter object that specifies criterias for the search. The default value is null, which means it will return all messages.
offset Optional. Set the offset for the search. The default value is 0, which means it will start the search from the first message. If either account or folder is null, the set value will be ignored.
limit Optional. Set the maximum number of results to return from the search. The default value is -1, which means all matching messages will be returned. If either account or folder is null, the set value will be ignored.
