Supported Platforms

Parameters
subject Optional. Indicates search to apply for message Subject field.
body Optional. Indicates search to apply for message Body field.
fromAddress Optional. Indicates search to apply for message FromAddress field.
recipient Optional. Indicates search to apply for message Recipient field.
read Optional. Indicates whether search to apply for read or unread messages. If specified, only read messages will be returned if the value is true; or only unread messages will be returned if the value is false.
flagged Optional. Indicates whether search to apply for flagged or unflagged messages. If specified, only flagged messages will be returned if the value is true; or only unflagged messages will be returned if the value is false.
today Optional. Indicates search to apply for messages that were received today. Only if specified and the value is true, only messages that were received today will be returned. If not specified or the value is false, it won't affect the search result.
yesterday Optional. Indicates search to apply for messages that were received yesterday. Only if specified and the value is true, only messages that were received yesterday will be returned. If not specified or the value is false, it won't affect the search result.
lastWeek Optional. Indicates search to apply for messages that were received last weekn. Only if specified and the value is true, only messages that were received last week will be returned. If not specified or the value is false, it won't affect the search result.

Properties

String subject


Indicates search to apply for message Subject field. By setting the subject, the user instructs the back-end to return only messages whose Subject field contain the specified value.
