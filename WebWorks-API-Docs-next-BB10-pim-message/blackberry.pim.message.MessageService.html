MessageService

blackberry.pim.message.Message create ([account : blackberry.pim.message.MessageAccount])


Returns a new blackberry.pim.message.Message object.


Supported Platforms
 - BlackBerry 10


Parameters
account Optional MessageAccount object that the message will use. If no account is specified, the message will be created in the default account.

Code Example: 
function test() {
    var email = blackberry.pim.message.MessageService.create(),
        MessageAddress = blackberry.pim.message.MessageAddress;

    console.log("email.status: " + email.status); // status is "new" now

    email.addresses = [new MessageAddress(MessageAddress.TO, "Eric", "eric@somedomain.com"),
                       new MessageAddress(MessageAddress.TO, "Sergey", "sergey@somedomain.com"),
                       new MessageAddress(MessageAddress.CC, "Hassan", "hassan@somedomain.com"),];
    email.subject = "Test";
    email.body = "Some body";
    email.addAttachment("attachment1", "text/plain", "myTextAttachment.txt");
    email.send(onSuccess, onError);

    console.log("email.status: " + email.status); // status is "sent" now

    var email2 = blackberry.pim.message.MessageService.create();

    console.log("email2.status: " + email2.status); // status is "new" now

    email2.subject = "Test";
    email2.body = "Some body";
    email2.save(onSuccess, onError);

    console.log("email2.status: " + email2.status); // status is "draft" now
}

void find (findOptions : blackberry.pim.message.MessageFindOptions, onFindSuccess : function, [onFindError : function])


Find messages from the message accounts based on some criterion.


Supported Platforms
 - BlackBerry 10


Parameters
findOptions Options to be applied to the search.
onFindSuccess Success callback function that is invoked with the messages found.

events: The array of blackberry.pim.message.Message objects from the search.
onFindError Optional error callback function. Invoked when error occurs.

error: The detailed error code when error occurs.
