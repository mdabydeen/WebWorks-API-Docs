document.createElement("video")


Supported Platform(s)

 - BlackBerry OS 7.0+
 - BlackBerry PlayBook 1.0+

Description

Creates an HTMLVideoElement



Code Example(s)

var video = document.createElement('video');
video.src = 'video.avi';
video.controls = true;
document.body.appendChild(video);
