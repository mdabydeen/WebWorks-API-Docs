ContactFindOptions

The ContactFindOptions object contains fields that filter and sort the results of blackberry.pim.contacts.find().

blackberry.pim.contacts.ContactFindOptions ([filter : Object[]], [sort : Object[]], [limit : Number], [favorite : Boolean])


Supported Platforms
 - BlackBerry 10


Parameters
filter Optional. An array of object literals that specifies search criteria to filter the results.

fieldName: The field being searched. This should be one of the SEARCH_FIELD_* constants.
fieldValue: The value of the field.
sort Optional. An array of object literals that specifies the order of the results.

fieldName: The field to sort. This should be one of the SORT_FIELD_* constants.
desc: The sort order (descending if true).
limit Optional. The maximum number of results to return from the search.
favorite Optional. If set to true, only Contacts marked as favorite would be included in search results.

Properties

Boolean favorite


If set to true, only Contacts marked as favorite would be included in search results.


Supported Platforms
 - BlackBerry 10

Object[] filter


An array of object literals that specifies search criteria to filter the results. The object should be in the following form: 

{
    fieldName: <field name, one of the SEARCH_FIELD_* constants>,
    fieldValue: <value of the field>
}


Supported Platforms
 - BlackBerry 10

Number limit


The maximum number of contact results returned. If limit is set to -1, all matching contacts will be returned.


Supported Platforms
 - BlackBerry 10

Object[] sort


An array of object literals that specifies the order of the results. The object should be in the following form: 

{
    fieldName: <field name, one of the SORT_FIELD_* constants>,
    desc: true to sort results in descending order     
}
