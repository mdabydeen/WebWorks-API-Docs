PushPayload

blackberry.push.PushPayload

blackberry.push.PushPayload (id : String, data : Blob, isAckRequired : Boolean)

Supported Platform(s)

 - BlackBerry 10

Description

This object cannot be created by the new keyword. It is generated only by the system on receipt of a push notification.



Parameter Type Description
id String The ID of a push.
data Blob The data of a push.
isAckRequired Boolean True if an acknowledgement is required for the push; false otherwise.
