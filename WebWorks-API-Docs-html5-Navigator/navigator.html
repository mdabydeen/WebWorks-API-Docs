offline


Supported Platform(s)

 - BlackBerry OS 6.0+
 - BlackBerry PlayBook 1.0+
 - BlackBerry 10
 - Ripple Emulator

Description

The offline event is triggered whenever browser switches to offline



Parameter Type Description
CallbackFunction function() The callback function that will be invoked on the offline event
useCapture Boolean A boolean that indicates whether the user wishes to initiate capture

Code Example(s)

function onOffline() {}
  document.body.addEventListener("offline", function() { alert("offline")}, false);
}
