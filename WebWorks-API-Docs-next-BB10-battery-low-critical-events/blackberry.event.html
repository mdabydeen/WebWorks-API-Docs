addEventListener

static void addEventListener(eventName : String, yourCallbackFunction : function)

Supported Platform(s)

 - BlackBerry 10

Description

Allows you to attach a callback function to an event. You can attach multuple callback functions to a single event.



Parameter Type Description
eventName String The event you want to attach the callback function to.
yourCallbackFunction function() The callback function that will execute when the event occurs.

removeEventListener

static void removeEventListener(eventName : String, yourCallbackFunction : function)

Supported Platform(s)

 - BlackBerry 10

Description

Allows you to remove a callback function from an event.



Parameter Type Description
eventName String The event you want to remove the callback function from.
yourCallbackFunction function() The callback function that will be removed from the event.

batterycritical

Supported Platform(s)

 - BlackBerry 10

Description

This event is fired by the system. If you want to listen to the event you can do so using the addEventListener function and remove the listener using the removeEventListener function.
The batterycritical event is triggered whenever the battery level changes to the value lower than 5%.



Parameter Type Description
yourCallbackFunction function(info : JSON) The callback function that will be invoked on the batterycritical event

info: An object the pertinent information
info.level: The percentage of the battery charge (0-100).
info.isPlugged: A boolean that represents whether or not the device is receiving a charge.

Code Example(s)

<script type="text/javascript">

function onBatteryCritical(info) {
  alert("The playbook " + info.isPlugged?"is ":"is not " + "plugged in with " + info.level + "% battery remaining");
}

blackberry.event.addEventListener("batterycritical", onBatteryCritical);

</script>

batterystatus

Supported Platform(s)

 - BlackBerry 10

Description

This event is fired by the system. If you want to listen to the event you can do so using the addEventListener function and remove the listener using the removeEventListener function.
The batterystatus event is triggered whenever the:

  • battery level changes
  • device starts to receive a charge
  • device stops receiving a charge



Parameter Type Description
yourCallbackFunction function(info : JSON) The callback function that will be invoked on the batterystatus event

info: An object the pertinent information
info.level: The percentage of the battery charge (0-100).
info.isPlugged: A boolean that represents whether or not the device is receiving a charge.

function onBatteryStatusChange(info) {
