Properties

static void create


Returns a new Contact object. This method does not persist the Contact object to the device contacts database. To persist the Contact object to the device, invoke the Contact.save method.


Supported Platforms
 - BlackBerry 10


Parameters
properties Optional object literal that specifies the field values for the Contact object. The object should be in the following form (with any number of properties): 
{
    displayName: <display name - String>,
    name: <name - ContactName>,
    nickname: <nickname - String>,
    phoneNumbers: <phone numbers - ContactField[]>,
    emails: <email addresses - ContactField[]>,
    addresses: <addresses - ContactAddress[]>,
    ims: <IM addresses - ContactField[]>,
    organizations: <organization - ContactOrganization[]>,
    birthday: <birthday - Date>,
    note: <note - String>,
    photos: <photos - ContactField[]>,
    categories: <user defined categories - ContactField[]>,
    urls: <web pages - ContactField[]>
}

Return:

blackberry.pim.contacts.Contact



Code Example: 
<script type="text/javascript">
    function onSaveSuccess(contact) {
        console.log("Contact with id=" + contact.id + " is saved!");
    }

    function onSaveError(error) {
        console.log("Error saving contact: " + error.code);
    }

    function createContact() {
        var contacts = blackberry.pim.contacts,
            ContactField = contacts.ContactField,
            name = new contacts.ContactName(),
            workPhone = new ContactField(ContactField.WORK, "123-456-789", true),
            workEmail = new ContactField(ContactField.WORK, "abc@blah.com", true),
            homeEmail = new ContactField(ContactField.HOME, "hello@me.com", false),
            contact;
        
        name.familyName = "Smith";
        name.givenName = "Joe";
        contact = contacts.create({
             "displayName": "Batman",
             "name": name,
             "phoneNumbers": [workPhone],
             "emails": [workEmail, homeEmail]
        });
        contact.save(onSaveSuccess, onSaveError);
    }
</script>

static void find


Queries the device contacts database. The search results are passed to the onFindSuccess callback function specified by the onFindSuccess parameter.


Supported Platforms
 - BlackBerry 10


Parameters
contactFields A String array of contact fields to be used as search qualifier. Only these fields will have values in the resulting Contact objects.
onFindSuccess Success callback function that is invoked with the contacts returned from the contacts database.

contacts: The array of Contact objects from the search.
onFindError Optional error callback function. Invoked when error occurs.

error: The ContactError object which contains the error code.
findOptions Optional object literal that describes the options to be applied to the search.

filter: An array of object literals used in order to select which contacts are returned. The object should be in the following form: 
{
    fieldName: <field name, one of the SEARCH_FIELD_* constants>,
    fieldValue: <value of the field>
}

limit: The maximum number of results to return from the search.
sort: An array of object literals that specifies how the results should be sorted. The object should be in the following form: 
{
    fieldName: <field name, one of the SORT_FIELD_* constants>,
    desc: true to sort results in descending order     
}

Code Example: 
<script type="text/javascript">
    function onFindSuccess(contacts) {
        console.log("Found " + contacts.length + " contacts in total");
        contacts.forEach(function (contact, index) {
             console.log("Contact " + index + ": " + contact.displayName);
        });
    }

    function onFindError(error) {
        console.log("Error: " + error.code);
    }

    function searchContacts() {
        var contacts = blackberry.pim.contacts;
        contacts.find(["displayName"], onFindSuccess, onFindError, {
             filter: [{
                 fieldName: contacts.SEARCH_FIELD_GIVEN_NAME,
                 fieldValue: "Joe"
             }],
             limit: 5,
             sort: [{
                  fieldName: contacts.SORT_FIELD_GIVEN_NAME,
                  desc: false 
             }, {
                  fieldName: contacts.SORT_FIELD_FAMILY_NAME,
                  desc: false
             }]
        });
    }
</script>
